# Musashi

Musashi is the online alias of Rohith, an Electronics and Communication student interested in computer architecture, RTL design, verification and SoC development.

Current areas of interest:

* Computer Architecture
* RTL Design
* Functional Verification
* Network-on-Chip Architectures
* FPGA-Based Systems

This blog serves as a collection of project logs, technical notes, design decisions and lessons learned while building digital systems.

## Current Projects

### BitFlip

A digital logic simulator built with C++ and Qt.

### Custom NoC

A packet-switched Network-on-Chip exploring routing, interconnect design and scalability.

### Multicore Processor Project

A long-term effort to build a multicore processor system around a custom NoC, including cache hierarchy, coherence and verification.
